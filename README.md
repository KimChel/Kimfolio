# Kimfolio

A personal portfolio website built with Next.js, featuring a pixel-art aesthetic with an interactive game engine scene. Visitors can explore an animated environment and navigate to sections covering my background, projects, skills, CV, and contact info.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Language:** TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.
